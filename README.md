# Custom-qr-code-generator

Generate your own qrcode in python. You can convert any link aur text into qrcode using python

first enter the link or the text in the ```url='here'``` 
then open the terminal and install the qr code module using:
```py
pip isntall -r r.txt
```
then run the file using:
```py
python main.py
```
Congratulations your custom qrcode is generated!
